# Cryptocurrencies

## Overview:
Client requested an analysis that includes what cryptocurrencies are on the trading market and how they could be grouped to create a classification system for this new investment based on the provided data. Data was cleaned and reformatted to be compatible with unsupervised machine learning. The python package sklearn was used to perform the machine learning and the package hvplot was used to present the data.


## Resources
- Source Data: crypto_data.csv
- Software: Python 3.10 (64-bit), Jupyter Notebook 6.4.8
